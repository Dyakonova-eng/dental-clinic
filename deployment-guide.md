## 1. Развёртывание сайта

### Вариант A: Бесплатный хостинг (Netlify)

1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Нажмите **«Add new site» → «Deploy manually»**
3. Перетащите папку `dental clinic` в окно загрузки
4. Сайт будет доступен по адресу вида: `https://your-site.netlify.app`
5. **Подключение домена:** Settings → Domain management → Add custom domain

### Вариант B: Хостинг с доменом (Recommended)

1. Купите домен (например, `smile-clinic.ru`) на [reg.ru](https://reg.ru) или [timeweb.com](https://timeweb.com)
2. Купите хостинг (от 100₽/мес)
3. Загрузите все файлы через файловый менеджер или FTP:
   - `index.html`
   - `style.css`
   - `script.js`
   - `quiz.js`
   - папку `images/`
4. Настройте DNS: домен → IP хостинга
5. Установите SSL-сертификат (бесплатный Let's Encrypt)

### Вариант C: GitHub Pages (бесплатно)

1. Создайте аккаунт на [github.com](https://github.com)
2. Создайте новый репозиторий: `smile-clinic`
3. Загрузите все файлы проекта
4. Settings → Pages → Source: `main branch`
5. Сайт доступен по адресу: `https://username.github.io/smile-clinic`

## 2. Настройка Telegram-бота

### Шаг 1: Создать бота

1. Откройте Telegram → найдите **@BotFather**
2. Отправьте команду: `/newbot`
3. Введите имя бота: `Smile Clinic Bot`
4. Введите username: `smile_clinic_crm_bot` (должен быть уникальным)
5. Получите **токен** — сохраните его! Выглядит как: `7123456789:AAHabcdefghijklmnopqrstuvwxyz`

### Шаг 2: Узнать свой Chat ID

1. Откройте Telegram → найдите **@userinfobot**
2. Отправьте любое сообщение
3. Бот ответит вашим **Chat ID** — сохраните его (число вида `123456789`)

### Шаг 3: Настроить бота (опционально)

Если хотите запустить полноценного бота с кнопками:

1. Установите [Node.js](https://nodejs.org) (LTS версия)
2. Откройте терминал в папке `automation/`
3. Выполните:
```bash
npm init -y
npm install node-telegram-bot-api
```
4. Откройте `telegram-bot.js` и вставьте:
   - `BOT_TOKEN`: ваш токен из шага 1
   - `OWNER_CHAT_ID`: ваш Chat ID из шага 2
5. Запустите: `node telegram-bot.js`

**Для постоянной работы** (на сервере):
```bash
npm install -g pm2
pm2 start telegram-bot.js --name "smile-clinic-bot"
pm2 save
pm2 startup
```

## 3. Настройка Make (Integromat)

### Шаг 1: Регистрация

1. Зарегистрируйтесь на [make.com](https://make.com)
2. Бесплатный план: 1000 операций/мес (хватит для старта)

### Шаг 2: Создание сценария

1. Нажмите **«Create a new scenario»**
2. Добавьте модуль: **Webhooks → Custom webhook**
3. Нажмите **«Add»** → дайте имя: «Smile Clinic — Заявки»
4. **Скопируйте URL webhook** — он понадобится для сайта!

### Шаг 3: Добавление модулей

Постройте цепочку по схеме из `make-scenario.json`:

```
[Webhook] → [Router] → [Telegram Bot] → [Google Sheets] → [Delay 24h] → [SMS]
                ↓
         [Горячий лид → срочное уведомление]
         [Тёплый лид → стандартное уведомление]
```

#### Модуль Telegram:
1. Добавьте модуль **«Telegram Bot → Send a Message»**
2. Вставьте токен бота
3. Chat ID: ваш Chat ID
4. В поле Message — скопируйте шаблон из `make-scenario.json`

#### Модуль Google Sheets:
1. Добавьте модуль **«Google Sheets → Add a Row»**
2. Подключите ваш Google-аккаунт
3. Выберите таблицу (создайте заранее, см. Шаг 4)
4. Настройте маппинг колонок

#### Модуль Router для классификации:
1. Добавьте **Router** после Webhook
2. Создайте 3 маршрута:
   - **Горячий:** `leadType = hot`
   - **Тёплый:** `leadType = warm`
   - **Стандартный:** все остальные

### Шаг 4: Тестирование

1. Нажмите **«Run once»** (внизу)
2. Отправьте тестовую заявку с сайта
3. Проверьте: Telegram, Google Sheets

## 4. Подключение Google Sheets

### Создание таблицы

1. Откройте [Google Sheets](https://sheets.google.com)
2. Создайте новую таблицу: **«Smile Clinic — Заявки»**
3. В первой строке создайте заголовки:

| A | B | C | D | E | F | G | H | I | J | K | L |
|---|---|---|---|---|---|---|---|---|---|---|---|
| Дата | Имя | Телефон | Услуга | Дата приёма | Время | Комментарий | Тип лида | Ожид. чек | Источник | Статус | Квиз |

4. Скопируйте ID таблицы из URL: `https://docs.google.com/spreadsheets/d/**ЭТОТ_ID**/edit`

## 5. Подключение webhook к сайту

### Вставка URL webhook

1. Откройте файл `script.js`
2. Найдите функцию `sendToWebhook`
3. Замените URL:
```javascript
const WEBHOOK_URL = 'https://hook.eu1.make.com/ВАШ_WEBHOOK_ID';
```
4. Раскомментируйте вызов `sendToWebhook(formData)` в функции `initBookingForm`

### В квизе (quiz.js):
1. Найдите функцию `sendQuizData`
2. Раскомментируйте `fetch(...)` блок
3. Вставьте тот же URL webhook

## 6. Настройка серии прогрева

Подробные шаблоны сообщений в файле: `automation/warmup-sequence.md`

### В Make:

1. Создайте отдельный сценарий: **«Smile Clinic — Прогрев»**
2. Триггер: **Schedule** (каждый час проверять новые подтверждённые записи)
3. Google Sheets: найти строки со статусом «Подтверждена»
4. Добавьте модули **Delay** + **WhatsApp/SMS** для каждого сообщения серии

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| Telegram-бот не отвечает | Проверьте токен, перезапустите |
| Make не получает данные | Проверьте URL webhook, CORS |
| Google Sheets не записывает | Проверьте права доступа таблицы |
| Сайт не отправляет заявки | Проверьте консоль браузера (F12) |
| SMS не приходят | Подключите SMS-сервис (SMS.ru, Twilio) |
