## Настройка в Make (Integromat)

### Сценарий 1: Брошенная заявка
```
[Google Sheets: проверка статуса] 
  → [Filter: статус ≠ "confirmed" AND дата > 48ч] 
  → [WhatsApp/SMS: Сообщение 1]
  → [Delay: 5 дней]
  → [Filter: статус всё ещё не "confirmed"]
  → [WhatsApp/SMS: Сообщение 2]
  → [Telegram: Уведомление администратору]
```

### Сценарий 2: No-show
```
[Schedule: каждый день в 21:00]
  → [Google Sheets: найти записи без отметки "visited" на сегодня]
  → [Filter: статус записи ≠ "visited"]
  → [WhatsApp/SMS: Сообщение 1]
  → [Telegram: СРОЧНО — no-show]
  → [Delay: 3 дня]
  → [WhatsApp/SMS: Сообщение 2]
```

### Сценарий 3: Спящий клиент
```
[Schedule: каждый понедельник]
  → [Google Sheets: найти клиентов с последним визитом > 6 мес]
  → [Iterator: для каждого клиента]
  → [WhatsApp/SMS: Сообщение]
  → [Google Sheets: обновить "дата последнего касания"]
```

## Метрики возврата

| Сценарий | Ожидаемый возврат | Средний чек возврата |
|----------|-------------------|---------------------|

### ROI автоматизации
- **Без автоматизации:** ~5% возвратов (только кто сам перезвонит)
- **С автоматизацией:** 20-35% возвратов
- **Дополнительный доход:** 50 000 – 200 000 ₽/мес (при 100+ заявках)
