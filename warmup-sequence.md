## Настройка в Make (Integromat)

### Сценарий: Warmup Sequence

1. **Триггер:** Webhook → получение подтверждённой записи
2. **Delay 1:** 24 часа → Сообщение 2 (памятка)
3. **Delay 2:** 3 дня → Сообщение 3 (совет)
4. **Router:** проверка услуги
5. **Delay 3:** 7 дней → Сообщение 4 (upsell — по услуге)
6. **Delay 4:** 14 дней → Сообщение 5 (повторный визит)
7. **Delay 5:** 30 дней → Сообщение 6 (отзыв)

### Переменные для подстановки:
- `{Имя}` → имя клиента из заявки
- `{дата}` → дата записи
- `{время}` → время записи
- `{услуга}` → выбранная услуга
